Notes about Crystal Beach DMA(CBDMA)

2023-05-14 Intel

Crystal Beach DMA(CBDMA)其实就是Intel® QuickData Technology，说白了就是offload memory copy to DMA engine，DSA代替了该技术。

FAST ‘23 paper Revitalizing the Forgotten On-Chip DMA to Expedite Data Movement in NVM-based Storage Systems也是使用了CBDMA来offload memory copy。
