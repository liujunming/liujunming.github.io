AWS Fargate vs AWS Lambda

https://stackshare.io/stackups/aws-fargate-vs-aws-lambda

  1. What is AWS Fargate? Run Containers Without Managing Infrastructure.
  2. What is AWS Lambda? Automatically run code in response to modifications to objects in Amazon S3 buckets, messages in Kinesis streams, or updates in DynamoDB.
  3. AWS Fargate can be classified as a tool in the “Containers as a Service” category, while AWS Lambda is grouped under “Serverless / Task Processing”.
