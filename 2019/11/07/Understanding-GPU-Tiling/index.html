Understanding GPU Tiling

2019-11-07 GPU

жң¬ж–Үе°Ҷдјҡд»Ӣз»ҚGPU tilingгҖӮ

еҲқе§ӢеҢ–жӯҘйӘӨпјҡ

  1. е®ҢжҲҗtiled region(дёҚеҗҢtiled regionд№Ӣй—ҙдёҚиғҪжңүoverlap)дёӯзү©зҗҶpageзҡ„еҲҶй…Қпјӣ
  2. е°Ҷtiled memoryжҳ е°„еҲ°ggttзҡ„logical address(fence regions must be aligned to a 4KB page)пјӣ
  3. е°Ҷfence regionзҡ„ејҖе§Ӣең°еқҖдёҺз»“жқҹең°еқҖзҡ„еҒҸз§»йҮҸ(иҝҷдәӣең°еқҖдҪҚдәҺggttдёӯ)еҶҷе…Ҙfence registerгҖӮ

CPUи®ҝй—®Tilingж—¶зҡ„жӯҘйӘӨпјҡ

  1. и®Ўз®—logical address ggttзҡ„offsetпјӣ
  2. жҜ”иҫғoffsetдёҺжүҖжңүfence registerдёӯејҖе§Ӣең°еқҖдёҺз»“жқҹең°еқҖпјӣ
  3. еҰӮжһңoffsetдҪҚдәҺжҹҗдёӘfence registerдёӯејҖе§Ӣең°еқҖдёҺз»“жқҹең°еқҖд№Ӣй—ҙпјҢйӮЈд№ҲиҰҒи®ҝй—®зҡ„еҶ…е®№жҳҜдҪҚдәҺtiled regionд№ӢеҶ…зҡ„пјӣ
  4. gpuж №жҚ®fence registerдёӯзҡ„дҝЎжҒҜпјҢе°Ҷlogical(linear) addressиҪ¬жҚўдёәtiled addressпјҢиҝҷж ·CPUе°ұиғҪи®ҝй—®tileдәҶгҖӮ
