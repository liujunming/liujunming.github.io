2019-08-17 I/O系统

文章目录

  2. 使用方法

本文主要介绍gpio的概念与使用方法。

概念

每个GPIO都代表一个连接到特定引脚的一个位。

“通用输入/输出口”(GPIO)是一个灵活的由软件控制的数字信号。

使用方法

gpio的确切含义由主板提供。在安装完gpio的驱动后，就可以在sysfs/class/gpio中使用了。

看完Linux GPIO access using sysfs后就能明白gpio的使用了。
