DPCD

DPCD(DisplayPort Configuration Data): a virtual memory with addresses 0x00000 - 0xFFFFF.

The message transfer between DP source and sink occur by reads and writes to the DisplayPort Configuration Data (DPCD) register of the sink device.

By reading certain registers in the DPCD, the source will be aware of the capabilities of the sink. During the process of establishing the data links, the Link Training, source writes to DPCD to indicate the target link configuration and sink in turn whites there the result of each link training stage.
